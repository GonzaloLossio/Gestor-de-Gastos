# Gestor de Gastos Personales

Aplicación web Full-Stack diseñada para la administración y control de finanzas personales. Este proyecto permite a los usuarios registrar, categorizar y visualizar sus gastos de manera eficiente.

##  Tecnologías utilizadas
* **Backend:** Python con Flask.
* **Base de Datos:** SQLAlchemy (SQLite/PostgreSQL) para la persistencia de datos.
* **Frontend:** HTML5, CSS3 y JavaScript (ES6+).
* **Interactividad:** Fetch API para actualizaciones asíncronas sin recargar la página.

##  Características
* Registro y edición de gastos en tiempo real.
* Categorización inteligente de egresos.
* Interfaz responsiva y dinámica.

##  Instalación y Uso
1. Clonar el repositorio: `git clone https://github.com/GonzaloLossio/gestor-gastos.git`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar la aplicación: `python app.py`
